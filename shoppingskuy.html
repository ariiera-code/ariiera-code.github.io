UI & Components

Design
Front Pages.

When buyers opens the website, some of them will not come back if the User Interface that is displayed the first time is not comfortable for them. So this experience will be vital especially in this part of the main page.

First, I offer an interesting visualization of product categories in the hero section. Below that I offer the newest, best-selling, and popular products. These 3 categories in my opinion will provide comfort to the user because to give trust to the user first. Minor visual details also contribute to this page, such as hover buttons & scrollbars

Login & Register UI

If you look at the current development of the UI, the login form is also an important aspect. Users feel safer if the visual UI that is applied follows the current trend. Of course the key trend now is animation.

For that I use animation in the textbox.

  • Textbox animation lights up when the user clicks the textbox
  • Placeholder animation goes up when the user clicks on the textbox

The purpose of applying the animation is so that when the user inputs data, they don't forget their purpose. for example, when inputting an email, but the user is confused whether this is an email/password input, for that I apply a placeholder animation above it
